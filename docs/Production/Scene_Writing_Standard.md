# Mandatory Rules

Include rules such as:

• Every Scene has exactly ONE Primary Purpose.

• Every Scene answers exactly ONE dramatic question.

• Every Scene advances the game.

• Every Scene must either:
    - Trigger exactly one Narrative Event
    OR
    - Prepare one future Narrative Event.

• Every Scene must reference:
    - Chapter Blueprint
    - Narrative Event
    - Story State(s)

• Every Scene must change at least one of:

Resource

Relationship

Knowledge

Time

Emotion

State

Variable

Player understanding

If none changes, the Scene must be removed.

• Every Scene ends with a reason to continue.

• Every Scene must preserve ambiguity.

• No Scene may confirm supernatural explanations.

• One cinematic image maximum.

• No filler dialogue.

• Dialogue must arise from conflict, objective or tension.

• Exposition must emerge naturally.

• Every choice must have purpose.

• Every meaningful choice must create consequence.

• Scene pacing must match its Primary Purpose.

• Scene length should be proportional to dramatic weight.
